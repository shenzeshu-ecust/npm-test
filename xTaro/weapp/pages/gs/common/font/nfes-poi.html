第一步：使用font-face声明字体 
  @font-face {font-family: 'nfes-poi';
    src: url('nfes-poi.eot'); /* IE9*/
    src: url('nfes-poi.eot?#iefix') format('embedded-opentype'), /* IE6-IE8 */
    url('nfes-poi.woff') format('woff'), /* chrome、firefox */
    url('nfes-poi.ttf') format('truetype'), /* chrome、firefox、opera、Safari, Android, iOS 4.2+*/
    url('nfes-poi.svg#iconfont') format('svg'); /* iOS 4.1- */
  }
第二步：定义使用iconfont的样式 
  .iconfont {
    font-family:"nfes-poi" !important;
    font-size:16px;font-style:normal;
    -webkit-font-smoothing: antialiased;
    -webkit-text-stroke-width: 0.2px;
    -moz-osx-font-smoothing: grayscale;}
